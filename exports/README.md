# Exports

В этой папке храним:
- экспортированные review images;
- annotated flow exports;
- handoff snapshots;
- placeholders для frame-групп, если сами `Figma`-исходники живут вне git.

## Правила
- имя файла должно отражать surface и flow, например `mob-settings-runtime-truth-v1.png`;
- экспорт не должен становиться единственным источником истины;
- рядом с export должно быть понятно, из какого backlog-артефакта он происходит.
